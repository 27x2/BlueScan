# BlueScan
A small program to scan hash, IP, and domain.

## Requests
  * Python 3
  ```# pip install -r requirements.txt```
  * API key of related services.
  
## Usage
Enter your API key of:
  * Virus Total
  * Abuseipdb
  * AlienVault
  * X-Force
  into related python file.
